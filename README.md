# 🔢 Contador utlizando listas em Python

## 📌 Descrição
Este projeto foi desenvolvido para praticar estruturas de repetição e condicionais em Python.

O programa percorre uma lista de números e conta quantas vezes um valor específico aparece.

---

## 🎯 Objetivo
Aplicar conceitos básicos de Python, como loops e condicionais, para resolver um problema simples de contagem de dados.

---

## ⚙️ Funcionalidades
- Receber um número digitado pelo usuário
- Percorrer uma lista de números
- Contar quantas vezes o número aparece
- Exibir o resultado na tela

---

## 🧠 Conceitos utilizados
- Listas em Python
- Estrutura de repetição (`for`)
- Estrutura condicional (`if`)
- Variáveis e contadores
- Entrada de dados com `input()`

---

## ▶️ Exemplo de execução

```bash
Entre com um número: 1
Quantidade de: 1 : 6
```
